# H.E.L.I.O.S
Custom Directory Brute-Force Tool by Alphabet Offensive Team.

## Cara Install
Untuk memastikan path wordlist tetap berada di `~/LKS/tool/` tanpa merubah script, jalankan perintah berikut:

```bash
git clone [https://github.com/anthrazst-ops/H.E.L.I.O.S.git](https://github.com/anthrazst-ops/H.E.L.I.O.S.git)
cd H.E.L.I.O.S
chmod +x setup.sh
./setup.sh
